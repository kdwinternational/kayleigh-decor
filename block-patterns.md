## Installation Instructions

1. **Copy each pattern code** into your theme's `patterns` directory or use the WordPress admin:
   - Go to **Appearance → Theme Editor**
   - Create new files: `home-pattern.php`, `category-pattern.php`, `product-pattern.php`
   - Paste the respective code

2. **Replace placeholders**:
   - `[CATEGORY_NAME]` with actual category name
   - `[CATEGORY_SEO_INTRO]` with the 80-word intro
   - `[CATEGORY_SLUG]` with the category slug
   - `[lead_time_days]` with the custom field value

3. **Upload placeholder images** to `/wp-content/uploads/` with the specified names

4. **Add custom CSS** to **Appearance → Customize → Additional CSS**:

```css
:root {
    --kayleigh-black: #111111;
    --kayleigh-beige: #D4B896;
    --kayleigh-white: #F5F5F5;
    --kayleigh-gold: #C9A227;
}

.has-kayleigh-black-background-color { background-color: var(--kayleigh-black) !important; }
.has-kayleigh-beige-background-color { background-color: var(--kayleigh-beige) !important; }
.has-kayleigh-white-background-color { background-color: var(--kayleigh-white) !important; }
.has-kayleigh-gold-background-color { background-color: var(--kayleigh-gold) !important; }
```
